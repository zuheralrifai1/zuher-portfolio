Layout & Grid System

A flexible, structured system that keeps all Elegant House layouts clean, balanced, and consistent across print and digital applications.

Base Grid System

The Elegant House layout system is built on a 12-column grid with generous margins and consistent spacing.
This grid ensures visual balance, guides alignment, and keeps all compositions clean and intentional.

  • 12 equal columns
  • 24px gutter between columns
  • 80–120px outer margins (left and right)
  • 80px top and bottom padding
  • Align all text and images to column boundaries
  • Avoid breaking the grid unless absolutely necessary

12-Column Base Grid

Left Image Layout

A strong layout for showcasing craftsmanship or product detail.
Use this when an image should draw immediate attention, supported by clear text.

Rules:

  • Align image to first 6 columns
  • Align text to remaining 6 columns
  • Maintain minimum 32px spacing between image and text

Hero Layout

Used for major announcements, introductions, or section openers.
The hero image spans all columns, with text placed below or overlaid depending on purpose.

Rules:

  • Hero spans all 12 columns
  • Title placed at 40–56px, centered or left-aligned
  • Maintain 48–72px spacing below the hero before placing text

Three-Column Grid

Ideal for listing services, features, or steps.
Provides balance and readability.

Rules:

  • Each column spans 4 grid columns
  • 24px gutter spacing maintained

Spacing & Alignment

Consistent spacing reinforces brand clarity and prevents visual clutter.
Follow these rules across all layouts:

Spacing Rules

  • Keep 72–96px spacing between major sections
  • Maintain 32px spacing between grouped elements
  • Avoid edge-to-edge content blocks
  • Always align elements to the 12-column grid
  • Keep text blocks between 5–7 columns for ideal readability

The Elegant House layout system creates structure, balance, and clarity across all brand applications.
By following this grid, every composition — from web pages to printed materials — remains refined, consistent, and visually harmonious.
