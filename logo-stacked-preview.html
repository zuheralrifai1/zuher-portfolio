Export commands (PowerShell)

# Inkscape (v1+)
inksape --export-type=png --export-width=1200 logo-stacked.svg -o logo-stacked-1200.png
# If Inkscape command is `inkscape.exe` on Windows, use full path or ensure it's on PATH

# ImageMagick (if it supports SVG on your system)
magick convert -background none logo-stacked.svg -resize 1200x logo-stacked-1200.png
